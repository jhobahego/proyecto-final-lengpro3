¿De que trata esta app?

Esta aplicación está hecha con el proposito de tener una aplicación sencilla que ayude a organizar y gestionar tareas diaras.

Con esta aplicación tienes un listado de tareas organizadas en una tabla, en la cual te puedes permitir crear tareas, editarlas, eliminarlas y marcar una tarea como completa, todo esto se almacenara en una base de datos.

La app cuenta con multiples ventanas con animaciones al realizar acciones que conllevan una consulta a base de datos, haciendo mas agradable la experiencia de usuario
